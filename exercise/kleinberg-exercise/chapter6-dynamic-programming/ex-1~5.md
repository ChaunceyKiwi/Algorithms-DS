### Exercise1
Find an independent set in a path G whose total weight is as large as possible.

#### Answer
a). With 6, 8, 6, 3, 6 : 8 + 6 < 6 + 6 + 6.

b). With 1, 8, 6, 3, 6 : Odd is 13, even is 11. Optimal is 14.

c). opt(n) = { opt(n-1), opt(n-2) + w(n) }
